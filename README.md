# Haxe + Nape + CreateJs

## Build & Dev

### Install deps

```
yarn
```

If fail to install haxe, delete `~/haxe`, and try `npm install haxeshim -g && npm install switchx -g && switchx`

### Set createjs-full to forked version:

```
yarn prepare
```

### Build

```
yarn build
```

### Dev

```
yarn dev
```
